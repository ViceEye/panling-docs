# 物品与装备

## 设计概述

物品系统涵盖装备、消耗品与功能道具三大类。装备按职业分类且不通用，消耗品以丹药体系为核心，功能道具支撑任务与系统交互。

## 装备分类

### 武器（按职业）

| 职业 | 分支一 | 分支二 | 通用副手 |
|------|--------|--------|----------|
| 战士 | 金钟武器 | 破军武器 | 防身短匕 |
| 弓箭手 | 游侠弓 | 狙击弓 | 防身短匕 |
| 炼丹师 | — | — | 防身短匕 |

### 防具

- **初心者套装**：新手基础防具，所有职业通用
- 后续防具通过锻造或任务获取

### 新手装备组合

每位新手玩家可兑换：
- 职业武器 ×1（对应所选分支）
- 防身短匕 ×1
- 初心者套装 ×1

## 消耗品

### 丹药体系

| 类型 | 作用范围 | 效果方向 | 说明 |
|------|----------|----------|------|
| 药丹 | 单人 | 正面 | 个人服用 |
| 法丹 | 多人 | 正面增益 | 群体增益 |
| 术丹 | 多人 | 负面/伤害 | 群体减益或伤害 |

### 丹药等级

| 等级 | 药引要求 | 元素倍率 | 效果强度 |
|------|----------|----------|----------|
| 初等 | 初级药引 | ×1 | 基础 |
| 中等 | 中级药引 | ×2 | 中等 |
| 高等 | 高级药引 | ×3 | 最强 |

### 关键消耗品

- **新手疗愈丹**：前期必不可少的回复物资，建议兑换一组
- **元素（金木水火土）**：炼丹核心材料
- **药引（初/中/高级）**：炼丹必需，较难获取

## 装备锻造

### 装备阶级

装备分为五个阶级，逐步提升：

| 阶级 | 说明 |
|------|------|
| 一阶 | 新手装备（初心者套装） |
| 二阶 | 第一次进阶装备 |
| 三阶 | 中期装备 |
| 四阶 | 后期装备 |
| 五阶 | 顶级装备 |

### 二阶装备制作

#### 制作条件

- 元素×32（其中至少10枚木元素）
- 铜钱×32
- 达到15级

#### 二阶武器制作

- 精炼木元素×2 + 职业武器雏形×1

#### 二阶防具制作

- 任意精炼元素×1 + 防具锻核×1 + 部位材料×1

#### 材料获取

- 精炼元素：铁匠铺兑换
- 武器雏形：铁匠铺兑换（消耗铜钱）
- 防具锻核：铁匠铺兑换（消耗铜钱）

### 装备套装属性

- 基础属性只需两件即可激活
- 职业专属与传说增幅须集齐四件同属性装备
- 五阶之前推荐土木/水木套装

## 货币系统

| 货币 | 说明 | 获取方式 |
|------|------|----------|
| 铜钱 | 基础货币，用于购买武器雏形与防具锻核 | 野外获取 |
| 银元 | 高级货币 | 山神庙任务奖励 |
| 赤铜锭 | 装备材料货币 | 怪物掉落 |

- 钱庄：各种族领地和皇城都有，用于货币存取
- 货币（铜钱、银元、赤铜锭）走数据库存储，不占背包格子

## 背包系统

- 默认背包容量：36格（原版默认）
- 物品堆叠上限：64（原版默认）
- 货币不占背包空间，通过数据库管理

## 邮件系统

当物品无法直接放入玩家背包时（背包已满），通过邮件系统暂存，玩家后续领取。适用场景：

- 任务奖励发放时背包满
- NPC 兑换产出时背包满
- 新手引导兑换时背包满
- 战斗中直接加入背包的奖励（非地面掉落物）

## 锻造台与炼丹炉交互

- 锻造台与炼丹炉均为**容器**（类似原版熔炉的交互模式）
- 材料放入容器后保留在容器中，直到玩家取出或炼制/锻造完成
- 掉线后材料保留在容器中不丢失
- 产出物留在容器中，玩家手动取出到背包

## 成长路线建议

- 龙须镇附近怪物强度较弱，适合新手发育
- 建议发育到拥有二阶装备后再前往青龙祭坛
- 推荐发育条件：元素×32 + 铜钱×32 + 重生石×5 + 新手疗愈丹×5 + 15级

## 实现要点

- 装备职业绑定检查
- 炼丹炉配方系统（5格、顺序验证）
- 锻造台交互（材料摆放、顺序检查）
- 道具消耗与补充逻辑
- 背包管理与堆叠系统
- 菜单书道具补充界面
